# dcs-mizreplace
A command line tool to update requiredModules in the <mission>.miz/mission file

WARNING: Always backup your mission file before using this tool!

Usage: mizreplace.exe <input.miz> <output.miz> [search] [replace]

input.miz is the incoming mission file, make a backup first!

output.miz is the outgoing mission file, overwritten if existing!

search is the term to replace in requiredModules of <input.miz>/mission file. If omitted the default is "Vietnam Assets Pack by EightBall & Tobi"

replace is the term that substitutes search in requiredModules in <output.miz>/mission file. If omitted the default is "[VWV] Vietnam Assets Pack"

WARNING: Always backup your mission file before using this tool!
