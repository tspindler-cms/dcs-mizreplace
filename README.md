# dcs-mizreplace
A command line tool to update requiredMods in the &lt;mission>.miz/mission file

Usage: mizreplace.exe <input.miz> <output.miz> [search] [replace]

input.miz is the incoming mission file, make a backup first!

output.miz is the outgoing mission file, overwritten if existing!

search is the term to replace in requiredMods of <input.miz>/mission file

replace is the term that substitutes search in requiredMods in <output.miz>/mission file
